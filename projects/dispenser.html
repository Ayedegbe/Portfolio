Automated Fluid Dispenser

This project involved designing and building a custom mechatronics system to automate the precise dispensing of fluids for a small-scale manufacturing process. The primary challenge was to eliminate inconsistencies and reduce material waste inherent in manual dispensing methods. I engineered a robust solution using SolidWorks for the custom enclosure and nozzle holder, ensuring secure mounting on the production floor.

The system's intelligence came from an Arduino microcontroller, programmed in Python, which precisely controlled a peristaltic pump and monitored fluid levels with a sensor. This resulted in highly accurate, repeatable dispensing, significantly reducing waste and improving production efficiency.

Tools & Tech Stack:

Mechanical Design: SolidWorks

Control/Programming: Arduino, Python

Components: Peristaltic Pump, Fluid Level Sensor, Custom Enclosure

Application: Precision Fluid Dispensing
