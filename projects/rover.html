Automatic Rover

This project involved developing an intelligent control system for an autonomous mobile robot designed to navigate and avoid obstacles. The core challenge was creating real-time decision-making capabilities to steer the rover without human intervention. I achieved this by integrating ultrasonic sensors for obstacle detection and a gyroscope for precise orientation. The system leveraged an A* path planning algorithm within LabVIEW to determine optimal routes and execute dynamic obstacle avoidance. This showcases my ability to bridge hardware with advanced control algorithms for truly autonomous functionality.

Tools & Tech Stack:

Control/Programming: LabVIEW

Algorithms: A* Path Planning

Sensors: Ultrasonic sensors, Gyroscope

Platform: Autonomous Rover (pre-built)
