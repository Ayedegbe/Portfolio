Loan Management System (Backend)

This project involved developing a robust Flask-based backend application for efficient loan management. The system automates critical financial operations, from client tracking to payment reminders and data reporting. It was designed to address the manual overhead and potential for error in traditional loan management processes, providing a secure and streamlined solution.

Key features include secure worker login, comprehensive client management, automated email reminders for upcoming and overdue payments, and seamless data export functionalities. The system integrates with PostgreSQL for reliable data storage and uses Flask-Mail for automated notifications, significantly improving operational efficiency and accuracy in financial tracking.

Tools & Tech Stack:

Backend Development: Python, Flask

Database: PostgreSQL

Automation: Automated Email Reminders (Flask-Mail), Data Export (CSV, Google Sheets)

Features: Secure Login, Payment Recording, Due Date Tracking
