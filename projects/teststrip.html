Automation for Diabetes Test Strip Manufacturing

This project was a comprehensive design and conceptualization effort focused on identifying opportunities for automation and efficiency improvements within a diabetes test strip manufacturing process.

The goal was to propose innovative mechatronic solutions to enhance production throughput, reduce errors, and optimize material handling. The project remained in the conceptual phase but laid a strong foundation for implementation.

Key design features included an automated servo-based spreading tool for consistent material application and a non-destructive resistance measurement system using roller probes and H-bridge circuitry. These tools were architected to be integrated into an existing production line under central PLC/HMI control.

Tools & Tech Stack:

Mechanical & System Design: SolidWorks

Precision Actuation: Servo Motor Control

Quality Testing: Non-Destructive Testing Principles (NDT)

Electrical Design: H-Bridge Circuit

Control Architecture: PLC + HMI (conceptual)
